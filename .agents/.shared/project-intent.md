# Project Intent: Infantex

## The Why
We are building a product for anxious first-time parents who need to trust
our platform with their infant's developmental milestones. Every design
decision must earn trust.

## Non-Negotiables
1. NEVER sacrifice perceived safety for visual flair
2. NEVER use dark patterns or urgency tactics
3. ALWAYS prioritize load speed (parents on mobile, often 3G)

## Current Stage: MVP
- Ship > Perfect
- Core features > Nice-to-haves
- Validated patterns > Novel experiments

## Trade-off Resolution Order
Security → Accessibility → Performance → UX → DX → Cost
